# 📋 Painel de Requisitos - Kit do Professor

O **Painel de Requisitos** é uma ferramenta interativa e educacional desenvolvida para apoiar professores e alunos de cursos técnicos de Desenvolvimento de Sistemas. O projeto funciona como um guia prático para as fases iniciais de conceção de software, unindo metodologias ágeis e engenharia de requisitos tradicional.

## ✨ Funcionalidades

O painel está dividido em 6 módulos essenciais para o arranque de qualquer projeto de software:

* **👥 Organização da Equipe:** Estruturação de grupos e delegação de papéis (Product Owner, UX/UI, Líder Técnico, etc.).
* **💡 Lean Canvas:** Ferramenta visual para validação da ideia de negócio e alinhamento do escopo.
* **📝 Product Backlog:** Mapeamento ágil utilizando Histórias de Utilizador (User Stories) e priorização em Sprints.
* **📖 Diário de Bordo:** Registo contínuo do progresso, comparando o planeado com o executado em cada aula.
* **⚙️ Casos de Uso (UML):** Esboço visual da interação entre os atores e a fronteira do sistema.
* **📄 Documento de Visão:** Template para especificação formal de requisitos e restrições não-funcionais.

## 🖨️ Otimizado para Sala de Aula

A aplicação foi desenhada a pensar no ambiente físico da sala de aula:
* **Modo "Modelo em Branco":** Transforma as ferramentas em folhas de exercícios pautadas.
* **Impressão Inteligente:** Conta com um botão de "Imprimir Projeto Completo" que compila todos os ecrãs num único PDF, alternando automaticamente entre a orientação de retrato e paisagem conforme o documento.
* **Tema Escuro/Claro:** Interface adaptável à preferência do utilizador para melhor leitura no ecrã.

## 🛠️ Tecnologias Utilizadas

* [React](https://reactjs.org/) - Biblioteca para construção da interface de utilizador.
* [Tailwind CSS](https://tailwindcss.com/) - Framework de CSS para estilização rápida, responsiva e suporte nativo a estilos de impressão (`@media print`).
* [Lucide Icons](https://lucide.dev/) - Ícones modernos e consistentes.
